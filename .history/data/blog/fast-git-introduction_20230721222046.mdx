# 概述

大部分和Git有关的文章都在告诉你Git有哪些命令，如何使用它们可以加速你的开发进程，但很少有人探讨Git的内部原理。

先考虑Git仓库的三个基本概念：Blob， Tree，Commits。

## Blob

以下是ChatGPT的总结：
- Blob是Git中存储文件内容的基本对象之一。
- 每个Blob对象由文件内容和SHA-1哈希值来标识。
- Blob对象与Git的树对象和提交对象一起构成了Git的三个基本对象类型。
